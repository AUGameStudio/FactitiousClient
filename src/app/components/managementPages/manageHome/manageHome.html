This page provides access to the tools for managing the Factitious game content. It will eventually be protected via a special username / password.

  • statistics - provides statistics for game play and article play.
  • game settings - allows changing the list of articles displayed and the scoring mechanisms, one a per round basis.
  • article editor - used to add and edit articles for use in the game.

You can also go to a special preview link to play the game while viewing it at various screen sizes. (This is useful during development to make sure that everything looks reasonable regardless of form factor.)
